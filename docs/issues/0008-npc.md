# Issue #8: NPCBase + NPCAvoidance（NPC 行走与躲避盲杖）

**依赖**: Issue #1, #5 (CaneSystem 的 Area3D)  
**参考**: PRD §2.7, §2.8, CONTEXT.md §NPC

---

## 需求

创建 NPC 基类和躲避行为模块。NPC 沿路径点连续移动，检测到盲杖靠近时自动避让。

## 范围

### NPCBase

1. 创建 `scripts/npc/NPCBase.gd`，`class_name NPCBase extends CharacterBody3D`
2. `@export var waypoints: Array[Node3D]`（路径点数组）
3. `@export var walk_speed: float = 2.0`
4. 循环遍历 waypoints，到达一个后转向下一个
5. 有 `CollisionShape3D`（盲杖碰到时有物理反馈）
6. 不参与地形惩罚

### NPCAvoidance

1. 创建 `scripts/npc/NPCAvoidance.gd`，`class_name NPCAvoidance extends Node`
2. 挂载在 NPC 下
3. 监听 CaneSystem Area3D 的 `area_entered` / `area_exited` 信号
4. 盲杖进入范围 → 暂停路径行走 → 横向平移一步（垂直于盲杖方向）→ 恢复行走
5. `@export var avoid_distance: float = 1.0`
6. 避让后冷却一段时间避免连续触发

## 验收

- NPC 沿路径点循环行走
- 盲杖靠近 NPC → NPC 往旁边让一步
- 盲杖直接碰到 NPC 身体 → 盲杖停住，有物理反馈
- 避让后 NPC 回到路径继续走
